About FemtoHTTPServer

FemtoHTTPServer is a educational HTTP server written entirely in the C# language and shipped as a single monolithic Windows executable. It supports serving static pages and CGI dynamic pages using CGI 1.1 gateway interface.

You can take part in the development of FemtoHTTPServer by controbuting on Github.

Note from the author

It have been a long time since I last touched Windows development, and back then my primary programming language was Visual Basic.net.
